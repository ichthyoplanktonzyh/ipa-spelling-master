# Phase 5.4: Local Record & Review

## 目标

提供纯浏览器、本地优先的录音与回放能力，让用户能听标准示范后录制并自我比较。

## 任务

- [ ] 5.4.1 定义录音身份、元数据和 repository 接口
- [ ] 5.4.2 实现 MediaRecorder 能力检测、录制、停止和资源清理
- [ ] 5.4.3 实现 IndexedDB 保存、读取、播放和删除 adapter
- [ ] 5.4.4 在 Guided Phoneme Lesson/训练浏览中加入可选录音卡片
- [ ] 5.4.5 提供隐私说明、权限拒绝与存储失败反馈
- [ ] 5.4.6 添加 repository、降级路径和组件交互测试

## 退出条件

- [ ] 支持的浏览器可录制、回放和删除单条本地录音
- [ ] 页面离开或录音停止后麦克风轨道被释放
- [ ] 权限拒绝、IndexedDB 失败或 MediaRecorder 缺失时训练仍可完成
- [ ] UI 明确说明录音不会上传，且不提供自动发音评分
- [ ] `npm run lint`、`npm run test` 和 `npm run build` 通过

## 影响文件

- `src/types.ts`

- `src/components/__tests__/*`

## 非目标

- 云端上传或跨设备同步
- 波形编辑
- ASR、Whisper 或 wav2vec2 发音评分
