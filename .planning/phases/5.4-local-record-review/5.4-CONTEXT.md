# Phase 5.4 上下文

## 来源

iSpeakerReact 使用 MediaRecorder 录制学习者发音，并在 Web 端通过 IndexedDB 保存音频。这个“录制—回放—自我比较”闭环可在不引入 ASR 的情况下增加产出练习价值。

## 关键决策

- 本阶段只做本地录音、回放和删除，不给出机器评分。
- 二进制录音使用 IndexedDB；`localStorage` 只保留轻量偏好或索引信息。
- 麦克风拒绝、浏览器不支持、存储失败都必须无损降级到现有听辨训练。
- 录音默认仅保存在用户设备，不上传、不分析。

## 约束

- 不引入 Electron、Python、FFmpeg 或模型下载。
- 不把录音结果计入 mastery 正确率。
- 必须清理 MediaStream、Blob URL 和播放资源。
