- `src/data/wordBank.ts`：英语词库迁移为 `TrainingItem[]`，去除 legacy `WordData` 运行格式，清理重复 identity，并规范少量 IPA ASCII 变体。

- `package.json`：新增 `npm run validate:data`。
- `.planning/codebase/*`、README：同步数据模型、测试策略、结构、架构、命令和词库统计。

## 偏离计划

## 遗留问题

- `src/components/IPAKeypad.tsx` 仍是未引用旧组件，建议后续清理。
- `scripts/validateData.ts` 目前没有 fixture-based 单元测试；后续建立 Vitest 时应补充校验器失败样例。
- `src/utils/judge.ts` 的 nearMatch 长度差异问题仍待 Phase 2.3 Feedback 阶段处理。

## 经验

- 数据校验脚本应和 profile 注册表、L1 映射注册表共用入口，避免新增语言后漏检。
- 拼音 canonical form 应区分“正字法 tone-number 字符串”和“内部音素 token 序列”；校验应验证两者的关系，而不是要求 round-trip 字符串完全相同。
