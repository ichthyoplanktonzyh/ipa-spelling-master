# Phase 5.5 上下文

## 来源

iSpeakerReact 提供 Dictation、Match-up、Reordering、Sounds and Spelling、Sorting、Odd one out、Snap 和 Memory match 等练习。PhoneticMaster 已有听音拼写和 minimal pair，因此不需要逐项复刻，而应提取可复用的题目、作答与计分结构。

## 关键决策

- 先实现共享 ExerciseDefinition/ExerciseSession 契约，再选择两种能直接服务声音范畴学习的练习。
- 首批候选为 Sounds and Spelling 与 Odd One Out；实施前用现有内容覆盖率确认最终顺序。
- 练习由 profile/content provider 提供数据，不在 React 组件里硬编码英语题目。
- 只有存在明确正确答案的练习才更新 mastery。

## 约束

- 不一次性复刻全部八种练习。
- 不引入通用低代码课程引擎。
- 不改变现有 `TrainingSession`/`MinimalPairSession` 的历史语义；如需新会话模型，应显式建模。
