# Phase 5.5: Data-driven Exercise Templates

## 目标

建立语言无关的练习模板基础，并交付两种新的声音感知练习，扩展单一拼写和 A/B 听辨之外的训练方式。

## 任务

- [ ] 5.5.1 盘点现有 Training/MinimalPair session 可复用部分与边界
- [ ] 5.5.2 定义 ExerciseDefinition、ExerciseQuestion、ExerciseSession 和 result read model
- [ ] 5.5.3 实现共享的题目调度、答案记录、计分和结果汇总工具
- [ ] 5.5.4 根据内容覆盖率实现 Sounds and Spelling 或等价练习
- [ ] 5.5.5 实现 Odd One Out 或第二种互补的感知练习
- [ ] 5.5.6 将有效结果接入 mastery 与下一步推荐
- [ ] 5.5.7 扩展数据校验、单元测试、组件测试和手动 QA 清单

## 退出条件

- [ ] 至少两种新练习使用同一 session/result 基础能力
- [ ] 英语和中文至少各有一种新练习可完成
- [ ] 新练习数据来自 profile/content provider，不含组件级语言分支
- [ ] 结果页能展示正确率、错题和下一步建议
- [ ] `npm run validate:data`、`npm run lint`、`npm run test` 和 `npm run build` 通过
