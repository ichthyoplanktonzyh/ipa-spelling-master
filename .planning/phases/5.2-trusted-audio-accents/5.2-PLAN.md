# Phase 5.2: Trusted Audio & Accent Variants

## 目标

为训练条目和最小对立体建立标准音频优先、TTS 安全降级、口音可选的统一播放层。

## 任务

- [ ] 5.2.1 定义语言无关的音频变体与口音/地区标签
- [ ] 5.2.2 实现统一 audio resolver 与播放状态机
- [ ] 5.2.3 接入普通训练、训练浏览、minimal pair 和音素详情播放入口
- [ ] 5.2.4 为少量许可明确的英语内容添加标准音频 fixture
- [ ] 5.2.5 提供口音选择与不可用时的透明降级提示
- [ ] 5.2.6 测试标准音频成功、加载失败、TTS fallback 和完全不可用路径

## 退出条件

- [ ] 有标准音频时不调用 TTS
- [ ] 标准音频失败时可自动回退且不中断当前题目
- [ ] 口音选择按 profile 声明生成，不含英语硬编码分支
- [ ] minimal pair 的标准音频/TTS 来源对用户可辨识
- [ ] `npm run validate:data`、`npm run lint`、`npm run test` 和 `npm run build` 通过

## 影响文件

- `src/types.ts`
- `src/profiles/*`
- `src/utils/voice.ts`
- `src/utils/audio.ts`（新增，名称可在实现时调整）
- `src/components/TrainingView.tsx`
- `src/components/MinimalPairView.tsx`
- `src/components/PhonemeDetailPanel.tsx`
- `public/audio/*`（仅许可明确的小规模资源）

## 非目标

- 完整离线媒体包
- 音频录制
- ASR 发音评分
- 用标准音频替换全部词库的 TTS
