# Phase 5.3: Guided Phoneme Lessons

## 目标

把音素详情升级为可完成的短学习路径，并在 profile 支持时展示位置例词、音节和重音信息。

## 任务

- [ ] 5.3.1 定义课程步骤 read model 与完成状态语义
- [ ] 5.3.2 聚合示范音频、位置例词、minimal pairs、L1 提示和练习入口
- [ ] 5.3.3 实现通用 Guided Phoneme Lesson 视图
- [ ] 5.3.4 为英语实现词首/词中/词尾与音节/重音的可选展示
- [ ] 5.3.5 为中文验证无位置分类时的自然降级
- [ ] 5.3.6 将课程完成动作接入现有 mastery/recommendation 但不虚构正确率
- [ ] 5.3.7 添加聚合工具和主要交互测试

## 退出条件

- [ ] 用户能从音素详情完成一次示范到专项练习的连续路径
- [ ] 英语可展示许可明确的位置例词和重音/音节信息
- [ ] 中文不出现英语专属标签或空白步骤
- [ ] 浏览课程不会被记为答题正确率
- [ ] `npm run lint`、`npm run test` 和 `npm run build` 通过
