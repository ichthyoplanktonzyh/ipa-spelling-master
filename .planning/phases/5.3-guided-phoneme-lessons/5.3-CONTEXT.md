# Phase 5.3 上下文

## 来源

iSpeakerReact 的 Sounds 体验把单个声音组织为示范、不同位置例词、练习素材和复习动作。PhoneticMaster 已有 `PhonemeDetail`、例词和 minimal pair 入口，但目前仍偏“资料抽屉”，缺少循序学习路径。

## 关键决策

- 在现有音素详情能力上深化，不另建一套与 Training Core 竞争的课程应用。
- 通用学习路径为：认识目标单元 → 听示范 → 听上下文例子 → 进入专项练习 → 查看下一步建议。
- “词首/词中/词尾”和“音节/重音”均为 profile 可选能力，不强加给拼音或未来语言。
- 课程内容缺失时继续展示现有详情，而不是隐藏音素入口。

## 约束

- 组件不直接读取语言数据文件。
- 不使用未经确认授权的发音说明、绕口令、图片或视频。
- 不在本阶段引入录音或发音评分。
