# Phase 5.6: New Language Pilot

## 目标

接入第 3 种目标语言，验证 Profile、可信内容、媒体和新练习能力的可选扩展边界。

## 任务

- [ ] 5.6.1 按内容许可、数据质量、音系复杂度和维护成本比较候选语言
- [ ] 5.6.2 记录选型决策并定义 canonical notation 与 display notation
- [ ] 5.6.3 实现 profile、parser、word bank 和注册入口
- [ ] 5.6.4 添加至少一组专项训练或新练习内容
- [ ] 5.6.5 验证无标准音频、无录音内容或无 L1 map 时的降级路径
- [ ] 5.6.6 扩展数据校验、parser/judge matrix 与主要 UI 测试
- [ ] 5.6.7 更新公开文档和语言接入规范

## 退出条件

- [ ] 新语言无需修改通用训练组件即可完成基础训练
- [ ] 每个难度档至少包含 20 条通过校验的训练条目
- [ ] 至少一种专项/新练习可完成并生成结果
- [ ] 所有内容来源和再分发状态可追溯
- [ ] `npm run validate:data`、`npm run lint`、`npm run test` 和 `npm run build` 通过
